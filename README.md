📌 Project Description

Movie Explorer is a web application that allows users to browse popular movies, search for movies, and save their favorite movies to a watchlist. The app uses a free movie API to fetch real-time data and display it in a user-friendly interface.

🚀 Features
View popular movies
Search movies by name
Add movies to watchlist
Remove movies from watchlist
Responsive and clean UI
Data fetched using API
🛠️ Technologies Used
HTML
CSS
JavaScript
TMDB API (The Movie Database)

🔑 API Used
TMDB API (https://www.themoviedb.org/
)
Used for fetching movie data like title, poster, and ratings
🎯 Learning Outcomes
Learned API integration using fetch
Improved JavaScript DOM manipulation
Built responsive UI using CSS
Understood localStorage for saving data
📸 Output
Displays movie list with posters
Search functionality works
Watchlist stores selected movies
